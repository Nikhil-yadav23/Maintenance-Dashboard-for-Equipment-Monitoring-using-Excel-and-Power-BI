# 📊 Maintenance Dashboard for Equipment Monitoring (Excel + Power BI)

## 🚀 Overview
This project is an interactive dashboard built using Excel and Power BI to monitor equipment performance, track failures, and analyze downtime.

## 🎯 Objective
* Reduce equipment downtime
* Improve maintenance planning
* Monitor performance efficiently

## 🛠️ Tools Used
* Microsoft Excel
* Power BI
  
## 📊 Dashboard Features
* Equipment performance tracking
* Downtime analysis
* Failure frequency monitoring
* Maintenance insights
